## ⛳️ Precourse

프리코스란? 온라인 코딩 테스트를 통과한 사람들을 대상으로 본 과정을 미리 경험해 보는 단계이다. 본 과정과 같이 매주 미션을 하나씩 구현하고 제출하는 방식으로 진행한다. 이 과정을 통해 지원자는 본 과정을 미리 경험해 보고 교육의 참여 여부를 결정할 수 있다.

 * 기간: `2021.11.24 ~ 2021.12.14`
